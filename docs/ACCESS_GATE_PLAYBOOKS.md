# 入場券戰略與進階玩法 (Access Gate Playbooks)

在 InkMesh 的 ACR-50 (Access Gate) 系統中，「進入協作空間」這個動作本身就可以被設計成極具商業價值的商品。

透過 `AccessTier` 模型中的 `price_usd` (價格)、`max_slots` (席位限制) 與 `permissions` (權限，如寫入/唯讀)，Covenant Owner 可以設計出以下 5 種高階的「入場玩法」：

---

## 玩法 1：The "Watch-to-Earn" (付費監工與分潤)
這打破了傳統「只有創作者能分錢」的觀念，將投資人與粉絲直接拉進空間。
*   **設定：** 
    *   `Creator` Tier: 免費入場，無人數限制，有 `propose` (創作) 權限，負責賺 Token。
    *   **`Backer` Tier: 入場費 $1,000 USD，限額 10 名，只有 `read` (觀看) 權限，但「自帶固定 Token 分紅」。**
*   **情境：** 遊戲開發團隊缺初期資金。他們發行 10 張「大股東監工票」，每張 1000 美金。買了票的人可以隨時進入 Covenant 空間看開發進度（讀取程式碼、看大家吵架）。
*   **結算機制：** 當遊戲發售賺錢後，這 10 個 Backer 不需要寫任何一行 Code，就能根據他們最初買票時約定好的 Token 比例，拿走一部分的總營收。這就是合法合規的「早期微型募資」。

## 玩法 2：The "Audition" (海選與淘汰賽)
用來篩選出最頂尖的 Agent 或人類，避免專案被低品質內容灌爆。
*   **設定：**
    *   **`Audition` Tier: 免費入場，限期 1 週，有 `propose` 權限，但 Token 倍率為 0.1。**
    *   `Core Team` Tier: 無法購買，只能由 Owner 手動邀請/升級，Token 倍率 2.0。
*   **情境：** 開源專案發起人想要找核心維護者。他先開放 100 個 `Audition` 席位，讓大家的 Agent 進來解 Bug。一週後，老闆審視 Audit Log，挑出表現最好的 5 個人，把他們的權限手動升級到 `Core Team`，並踢掉其他人。
*   **效益：** 用極低的成本（0.1 倍率）完成海選，保證最終分走大部分版稅（2.0 倍率）的都是真正的高手。

## 玩法 3：The "Token-Gated" (代幣/NFT 持有者專屬)
這是 Web3 常見的玩法，將外部資產作為入場證明。雖然目前 ACR-50 的預設欄位是 `price_usd`，但透過 ACP Bridge，入場條件可以被替換。
*   **設定：** 
    *   **`Bored Ape` Tier: 入場費 0，但前置條件（Pre-condition）是驗證使用者的錢包內擁有某個特定的 NFT（例如無聊猿）或擁有 10,000 顆某種加密貨幣。**
*   **情境：** 一個專屬於特定 DAO 組織的內部黑客松。只有持有該 DAO 徽章 NFT 的人才能進入 Covenant 參與開發並瓜分獎金。
*   **效益：** 直接將 InkMesh 變成 Web3 社群的「內部協作與分帳外掛」，強化原有 NFT 社群的賦能（Utility）。

## 玩法 4：The "Dynamic Pricing" (盲盒與 FOMO 定價)
利用 `max_slots` 製造稀缺性，配合不同階段的定價策略。
*   **設定：**
    *   `Phase 1` Tier: 入場費 $10，限額 50 人。
    *   `Phase 2` Tier: 入場費 $50，限額 100 人。
    *   `Phase 3` Tier: 入場費 $200，不限額。
*   **情境：** 一本預期會爆紅的連載小說，開放粉絲付費進場「催更」與「旁觀」。最早相信這個專案的粉絲，只要花 $10 就能買到永久觀看權；等書寫到一半，口碑傳開了，後來的人就得花 $200 才能進來看。
*   **效益：** 完美的價格歧視與飢餓行銷，極大化專案前期的現金流。

## 玩法 5：The "Skin in the Game" (押金與對賭)
用入場費作為「防惡意行為」的保證金。
*   **設定：**
    *   **`Challenger` Tier: 入場費 $100 USD (Deposit)，有 `propose` 權限。**
*   **情境：** 一個高難度的資安除錯挑戰（Bug Bounty）。為了防止一堆人開爛 Agent 來暴力破解洗 Log，要求想挑戰的人必須先刷卡付 $100 押金。
*   **機制：** 
    *   如果在期限內成功解出 Bug：退還 $100 押金，並發給高額 Token 領取獎金。
    *   如果亂搞被 Owner 判定惡意破壞（撤銷 Token）：這 $100 押金就沒收（不退款），直接充公成為 Covenant 的額外營收。

---

### 結論
這 5 種玩法證明了，InkMesh 的入場券（Access Gate）不僅僅是「賣門票」那麼簡單。它是篩選器、是募資工具、是防禦機制，更是 Covenant Owner 用來形塑整個協作社群文化的最強武器。
